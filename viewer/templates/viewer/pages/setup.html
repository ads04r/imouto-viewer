Welcome to Imouto Viewer

Imouto Viewer is a system for managing and displaying lifelogs. In order to use it, some additional setup is required.

Imouto Location Manager

The main functionality of the viewer requires an instance of Imouto Location Manager to be running. Please install this, and ensure the LOCATION_MANAGER_URL variable in settings_local.py points to the location at which this may be accessed.

Events

For the viewer to operate, you need to create one or more events. These are generally things than happen to you; visits to places, journeys, exercise, or social occasions. However, to start things off, Imouto can create an "Installed Imouto Viewer" event automatically by clicking the button below.
